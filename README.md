## Docker image

```
https://hub.docker.com/r/christianbillp/sensor_network/

```

## mongo.conf format
```
[conf]
username=USERNAME
password=PASSWORD
hostname=HOSTNAME
port=PORT (default is 27017)
db=DATABASE
```
